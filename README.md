Instalar Angular
````
npm i -g @angular/cli@14.2.12 --force
````
Clonar el repositorio
````
git clone https://github.com/UC-Programacion-3-2023-2/template-angular.git frontend
````
Acceder a la carpeta "frontend"
````
cd frontend
````
Instalar las dependencias
````
npm install --force
````
Configurar API Key de Gemini
````
1. Obtener API Key en: https://aistudio.google.com/app/apikey
2. Editar src/environments/environment.ts
3. Reemplazar 'TU_API_KEY_AQUI' con tu API Key
````
Correr el servidor
```
ng serve
```
